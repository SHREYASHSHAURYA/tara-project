# TARA – AI Loan Assistant
A complete loan‑assistant system with:

- FastAPI backend
- Interactive chat UI
- Login + Signup system
- Session handling (localStorage)
- Dark/Light mode
- Glass UI + animations
- Logout system
- PDF sanction letter generation
- Full underwriting workflow
